1. Как посмотреть содержимое текстового файла?
	less <filename>
	cat <filename>
	head <filename>
	tail <filename>
2. Что делает команда tail? Как посмотреть?
	Команда «tail» выводит последнюю часть (по умолчанию 10 строк) каждого
	файла. Чтобы узнать, что делает команда, нужно ввести команду «info tail» или «man tail».
	tail <filename>.

11. Как создать файл /tmp/2mb.txt размером 2Mb?
	1) fallocate -l <size_of_file> filename
	2) truncate -s <size_of_file> filename
	3) dd if=/dev/zero of=<filename> bs=<filesize> count=<number>
		if=FILE - read from file instead of standart input
		of=FILE - Write to FILE instead of standart input...
		bs=BYTES- Set both input and output block sizes to BYTES.
			  This makes 'dd' read and write bytes per block
			  ovveriding and 'ibs' and 'obs' settings.
		count   - Copy N 'ibs' -byte blocks from the input file,
			  instead of everything until the end of the file.
	4) head -c <filesize> /dev/zero > <filename>
	5) xfs_mkfile <filesize> filename
12. Как узнать тип файла?

13. Как переименовать файл?

14. Как удалить файл/директорию?
	удалить файл: rm <filename>
	рекурсивное удаление директории с файлами и вложенными директориями: rm -r <dir_name>
	удаление пустой директории с вложенными пустыми директориями: rmdir -p <dir_path_name>
15. Как создать символическую/жесткую ссылку на файл/директорию?
	softlin: ln -s <source_filename> <linkname>
	hardlink: ln -s <source_filename> <linkname>
16. Как посмотреть размер файла?

17. Как как узнать размер директории?
	размер директории: ls -sh <dirname>
	размер директории: du -sh <dirname>

19. Как посчитать количество строк в текстовом файле?

20. Как вывести на экран отсортированные строки текстового файла?
